Designed and implemented a secure, scalable Healthcare Appointment Scheduling System that modernizes how patients, doctors, and administrators manage healthcare appointments. The system replaces inefficient manual booking with a robust, real-time digital platform that minimizes no-shows, improves scheduling efficiency, and protects sensitive medical data.

The backend is powered by Flask and SQLAlchemy ORM, supporting complex relational data models for patients, doctors, appointments, and medical records. OAuth 2.0 with Google enables seamless new patient registration, while JWT-based Role-Based Access Control (RBAC) enforces secure access across patients, doctors, and admins. All API endpoints are protected with token validation, CSRF safeguards, and encrypted communication.

Patients can securely book, cancel, or reschedule appointments; doctors manage availability and medical records; and administrators oversee system operations. Notifications are automated via Mailgun, and the system integrates optional Google Calendar APIs for scheduling sync. Built with modular object-oriented design, the project also includes a custom CLI tool (console.py) to manage users, appointments, and records directly from the terminal.
