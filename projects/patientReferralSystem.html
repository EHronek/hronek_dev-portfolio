Patient Referral System

  • Tech Stack: Python, Flask, SQLAlchemy, ReactJS, MySQL, Docker
  • Github URL: Project Link

Built a secure and scalable Patient Referral System that streamlines the process of referring patients between healthcare providers. The system provides role-based access control (RBAC) across multiple user types (doctors, referral staff, specialists, and patients) to ensure security and accountability in healthcare workflows.

Doctors can initiate referrals, referral staff process and assign cases, specialists accept or reject referrals and schedule appointments, and patients receive real-time notifications via SMS and email. All key actions (referrals, approvals, scheduling) are logged through an Audit Log for traceability and compliance.

Integrated external services such as KenyaEMR for medical history retrieval, DHIS2 for referral statistics reporting, and Twilio/Africaâ€™s Talking for SMS and email notifications. The backend is built with Flask and SQLAlchemy, while the frontend uses ReactJS for a responsive user experience. Containerized with Docker for ease of deployment.

Key Features:

  • Role-based access control (RBAC) for different user groups.
  • Secure referral creation, processing, and appointment scheduling workflows.
  • Real-time patient and doctor notifications (SMS/email).
  • Audit logging of all critical actions for accountability.
  • External integrations with KenyaEMR and DHIS2.
  • Scalable backend with Flask + SQLAlchemy and a ReactJS frontend.
