Designed and developed the official website for The Gretchensville Home, a boutique hospitality brand focused on comfort, warmth, and personalized guest experiences. The project combines a polished frontend with a secure, production-grade backend API to support real client communication workflows.

The frontend was crafted with a clean, elegant, and welcoming design, reflecting the homely yet professional nature of the property. The layout emphasizes intuitive navigation, visual storytelling, and responsiveness across all devices, ensuring a seamless user experience for prospective guests.

A key feature of the website is a fully integrated contact form connected to a custom backend built using Django REST Framework. When a visitor submits an inquiry, the system securely processes the request and:

  • Sends a confirmation email to the client acknowledging receipt of the message.
  • Triggers an internal notification email to staff, ensuring prompt follow-up.

The backend API was developed with a security-first mindset, implementing strict input validation, request sanitization, and defensive controls to protect against abuse and malformed data. To ensure reliability and trust, additional safeguards were applied, including:

  • API rate limiting to prevent spam and automated abuse.
  • Cross-Origin Resource Sharing (CORS) configuration for controlled frontend access.
  • Robust validation rules to enforce data integrity.
  • Secure email handling and error management.

This approach ensures that both guests and staff benefit from a reliable, secure, and professional communication channel, while maintaining operational efficiency and protecting system resources.
